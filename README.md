## 🎯 프로젝트 소개

React와 Typescript 를 사용한 맛집 정보 사이트

✨ 주요 특징
- 전체 맛집 정보 가져오기
- 현재 위치와 가장 가까운 맛집 순서로 표시하기
- 맛집 찜하기
- 찜한 맛집 불러오기
- 찜한 맛집 삭제하기
